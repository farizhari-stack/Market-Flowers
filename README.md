# Stefii's Store

Welcome to the official repository for **Stefii's Store** landing page. This project showcases a premium, modern web experience for a boutique store specializing in high-end collections, wedding services, and corporate gifts.

## 🌟 Features

- **Modern UI/UX**: A clean, elegant design built with a focus on aesthetics and user experience.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop views.
- **Premium Components**: Includes sections for Collections, Weddings, Corporate Gifts, Workshops, and more.
- **Interactive Animations**: Smooth transitions and scroll effects powered by `motion`.
- **Dynamic Testimonials**: Animated feedback from satisfied customers.

## 🛠️ Technologies Used

- **React 19**: Core frontend framework.
- **Vite**: Ultra-fast build tool and development server.
- **Tailwind CSS**: Utility-first CSS framework for modern styling.
- **Motion**: High-performance animation library.
- **Lucide React**: Clean and consistent icon set.
- **TypeScript**: Static typing for robust code development.

## 🚀 Getting Started

Follow these steps to set up and run the project locally on your machine.

### Prerequisites

- [Node.js](https://nodejs.org/) (Latest LTS version recommended)
- npm (comes with Node.js)

### Installation

1. **Clone the repository:**
   ```bash
   git clone [repository-url]
   cd stefiisStore
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

4. **Build for production:**
   ```bash
   npm run build
   ```
